# Frontend Template

Reusable React, TypeScript, Vite, and CSS variables template for the video-community UI skill.

## Run

```bash
npm install
npm run dev
npm run build
```

## Adapt

- Replace `src/data/fixtures.ts` with neutral or user-provided content.
- Map `design-tokens.json` values into `src/styles/tokens.css`.
- Keep loading, empty, error, and success branches visible during review.
- Keep `focus-visible` and reduced-motion behavior when changing components.
