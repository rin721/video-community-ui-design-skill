# Forward Test: Static HTML

Use $video-community-ui-design-skill at this repository to create a zero-dependency static HTML/CSS/JS media discovery page.

The page must open directly, include neutral fixtures, show decorative masthead, category tabs, announcement bar, media feed, loading/empty/error/success states, mobile top and bottom navigation, focus-visible, and reduced motion.

Do not describe the static page as MUI. If a prompt requires Material UI, switch to the React Material UI forward test instead of imitating the library in static HTML.
