# Forward Test: Static HTML

## Prompt

Use $video-community-ui-design-skill to create a zero-dependency static HTML/CSS/JavaScript video-community landing page. Use the static template asset guidance. Include semantic HTML, CSS variables, media cards, filters, loading/empty/error/success states, keyboard-visible focus, responsive behavior, and direct-open instructions.

## Expected Acceptance Points

- Output can run as static files without a framework.
- CSS includes token variables, focus-visible, responsive rules, and prefers-reduced-motion.
- JavaScript keeps state handling small, readable, and independent of real services.
