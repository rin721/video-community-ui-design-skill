# Forward Test: Design Spec

## Prompt

Use $video-community-ui-design-skill to create a UI design spec for a light video-community discovery homepage. Include token usage, layout grammar, media cards, search filters, hover/focus/loading/empty/error states, desktop/tablet/mobile behavior, and WCAG AA notes.

## Expected Acceptance Points

- Output references semantic tokens instead of hard-coded visual values.
- Output includes layout, component anatomy, states, responsive rules, accessibility, and release hygiene checks.
- Output avoids real brands, real people, real business data, and fixed non-replaceable copy.
